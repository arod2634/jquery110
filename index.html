Welcome to jQuery 110

jQuery is a JavaScript library supported by all major web browsers that simplifies creating rich, web-based user interfaces. It simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. In this hands-on course students learn how to use the jQuery UI library to develop captivating user interfaces for their Web sites and applications and how to simplify client-side scripting tasks.

Topics include:

  • jQuery Selectors
  • DOM traversal and modification
  • Animation Effects
  • Interaction Helpers
  • Interface components like Datepickers and Accordions
  • How to use plugins to extend jQueryâ€™s capabilities
  • How to make asynchronous JavaScript and XML calls
